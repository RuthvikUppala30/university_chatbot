About the Project

Your Personal Assistant for University Queries

This Chatbot system is designed to address a wide range of queries for all types of users such as students, employees, faculty, and parents. It can handle inquiries related to Admissions, Course Registration, On-Campus Events, Employment Opportunities, and more.

Using Natural Language Processing (NLP) and Machine Learning Algorithms, this chatbot is capable of understanding user inputs and providing relevant responses. It can be seamlessly integrated into the university's existing system, and has features like scheduling appointments, checking application statuses, and retrieving updated information from a connected database.

The primary objective is to enhance user experience, save time, reduce IT department workload, and increase operational efficiency.
